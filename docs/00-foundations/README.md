# Module 01 — AI Foundations

---

## Objective

Build a strong conceptual foundation for modern AI systems before diving into GPUs, CUDA, inference engines, and platform engineering.

This module focuses on understanding the complete lifecycle of Large Language Models (LLMs), from training to production deployment.

---

## Learning Outcomes

After completing this module you should be able to explain:

- The AI ecosystem
- The LLM lifecycle
- Training vs Fine-tuning vs Inference
- Model Serving
- Evaluation
- Tokenization
- Embeddings
- Retrieval-Augmented Generation (RAG)
- AI Agents
- Mixture of Experts (MoE)
