# Atlas

> **An open engineering handbook for designing, building, benchmarking, and operating AI infrastructure—from a single GPU to a production-scale inference platform.**

---

## 🚀 Vision

Atlas is a long-term engineering project focused on understanding and building modern AI infrastructure from first principles.

The project has three complementary goals:

- 📚 Build a comprehensive AI Infrastructure knowledge base
- 🏗️ Build a production-quality AI inference platform
- 💼 Build a foundation for a future AI infrastructure business

Rather than only learning technologies, Atlas focuses on understanding why they exist, how they work internally, how they are used in industry, and how to apply them when building real-world systems.

---

## Project Goals

Atlas aims to answer questions like:

- How do companies like OpenAI, Together AI, RunPod and CoreWeave build AI infrastructure?
- What happens inside a GPU while serving an LLM?
- Why is CUDA required?
- Why are inference engines like vLLM and SGLang different?
- How does Kubernetes schedule GPUs?
- How do we design a scalable inference platform?
- How do we operate such a platform efficiently and economically?

## Engineering Principles

Atlas is guided by a few core principles.

- Learn from first principles.
- Build whenever practical.
- Benchmark instead of guessing.
- Document engineering decisions.
- Prefer open technologies.
- Think like an AI infrastructure engineer.

## Long-Term Vision

Atlas will gradually evolve into:

- An AI Infrastructure Handbook
- A production-ready inference platform
- A collection of engineering benchmarks
- A portfolio demonstrating AI infrastructure engineering
